# The-bigs-3

This is an Arcade Browser Baseball game.
A multiplayer browser baseball game inspired by The Bigs 2.

## Vision
Fast, smooth, modern, arcade-style baseball in the browser.
Stylized visuals, short matches, competitive online play.

## MVP
One networked at-bat:
- pitcher selects pitch type + target
- batter reacts and swings
- server resolves outcome
- client visualizes result

## Current constraints
- Mobile-first setup and planning
- Unity client to be built later on desktop
